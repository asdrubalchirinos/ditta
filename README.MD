# Ditta Codex Pet

This repository contains a minimal Codex Pet package for **Ditta**.

## Required files

- `pet.json` — pet metadata and config
- `spritesheet.webp` — animated pet spritesheet used by Codex

## How to use this pet in Codex

1. Keep `pet.json` and `spritesheet.webp` together in the same folder.
2. Open Codex and go to the pet selection/import flow.
3. Import this pet folder (or `pet.json`, depending on your Codex setup).
4. Select **Ditta** as your active pet.
